# Echo的柔术记录 — 产品设计文档

## 1. 产品概述

**类型：** 微信小程序
**用途：** 巴西柔术训练记录工具，支持记录实战表现、招式、心得、训练流程

---

## 2. 数据模型

本地存储（7个数据实体），全部使用 `wx.setStorageSync`。

| 类型 | 存储键 | 说明 |
|------|--------|------|
| 实战记录 | `sessionRecords` | 训练课状态、疼痛标签、实战表现 |
| 招式 | `techniques` | 招式库，含位置/动作标签、招式解说、视频链接 |
| 心得 | `insights` | 训练心得/反思 |
| 流程 | `flows` | 树形训练流程图，含分支、条件 |
| 自定义招式位置标签 | `customTechTags.positions` | 用户自定义 |
| 自定义上课状态标签 | `customSessionClassStates` | 用户自定义 |
| 自定义疼痛部位标签 | `customSessionPainTags` | 用户自定义 |

### 核心字段

**实战记录**
- `id`, `recordDate`, `classState`, `painTags[]`, `painDescription`, `practiceNotes`, `createdAt`, `updatedAt`

**招式**
- `id`, `name`, `position`, `action`, `notes`, `teachVideoLinks[{url, title}]`, `recordDate`, `createdAt`, `updatedAt`

**心得**
- `id`, `title`, `body`, `recordDate`, `createdAt`, `updatedAt`

**流程**
- `id`, `name`, `startPosition`, `startAction`, `nodes[]`, `edges[]`, `rootId`, `notes[]`, `recordDate`, `createdAt`, `updatedAt`

## 3. 页面结构

### 底部 Tab 导航（3个）
- **首页** — 最近记录列表 + 快捷入口
- **新建** — 三个页签创建不同类型记录
- **历史** — 日历视图 + 统计 + 导入导出

### 首页 (`/pages/index`)
- 顶部标题 "Echo的柔术记录" + "开始今日记录"按钮
- 最近记录列表，支持：
  - 时间范围筛选（全部/本周/本月/自定义）
  - 关键词搜索
  - 记录类型+招式标签筛选
- 长按删除记录

### 新建页 (`/pages/add`) — 三个页签

| 页签 | 内容 |
|------|------|
| 实战表现 | 上课状态标签（单选）、疼痛标签（多选）、疼痛描述、实战表现（可展开全屏编辑） |
| 新建招式 | 招式名称、位置、动作类型、招式解说（可展开）、教学视频链接（支持粘贴自动提取标题和链接） |
| 新建心得 | 标题、正文（可展开全屏编辑） |

- 所有文本框右侧有"展开"按钮，展开后顶部右上角显示"完成"
- 支持自定义标签（＋按钮弹出对话框）
- 提交后显示保存成功提示

### 历史页 (`/pages/history`)

两个 Tab：
- **日历视图** — 按月展示每天的记录数量，点击展开当日记录列表（只读）
- **统计** — 训练天数、招式/心得数量；按位置/动作分组的招式列表；导出 TXT / 备份恢复

### 详情页（只读，展示+编辑入口）
- `sessionDetail` — 实战记录详情（已移除复制按钮）
- `techDetail` — 招式详情，视频链接点击跳转 webview
- `insightDetail` — 心得详情
- `classDetail` — 流程详情，内联招式标签点击弹出简介

### 其他页面
- `addTech` — 独立招式编辑页（编辑按钮进入）
- `flowEditor` — 流程树编辑器（最多3分支、3层深度）
- `review` — 按时间/标签筛选复习招式
- `webview` — 外部链接页；加载失败时显示"复制链接"提示而非报错

## 4. 标签体系

**位置**（单选）：封闭式防守、半开放式防守、开放式防守、半防守、侧压、骑乘、北向南、膝压、背后控制、站姿、乌龟、过腿

**动作类型**（单选）：降服、逃脱、过腿、扫倒、摔、拉开、控制

**上课状态**（单选）：状态很好、尚可、一般、疲惫、恢复中

**疼痛部位**（多选）：无、脚、腿、肘、颈、躯干

以上均可通过＋按钮自定义添加。

---

## 5. 核心功能

- **全屏展开编辑** — 三个文本区域（实战表现/招式解说/正文）右上角有展开入口，展开后左上角关闭，右上角完成
- **视频链接粘贴** — 自动从粘贴文本中提取 URL 和显示名称
- **搜索** — 跨类型搜索名称、标签、备注
- **数据导出** — TXT 格式训练数据分享；JSON 全量备份（含选择性恢复预览）
- **编辑功能** — 各详情页右上角"编辑"按钮，跳转新建页并带入数据
- **业务域名限制** — 小程序 webview 只能打开已配置域名；无法打开的链接（小红书/Bilibili等）提示用户复制到聊天中打开
